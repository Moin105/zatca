# ZATCA E-Invoicing Frontend

Next.js frontend application for the ZATCA Phase 1 E-Invoicing System.

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- Company management
- Customer management
- Invoice creation and management
- Invoice issuance with QR codes
- PDF and XML generation

## API Configuration

Set the `NEXT_PUBLIC_API_URL` environment variable to point to your backend API (default: http://localhost:3001)
